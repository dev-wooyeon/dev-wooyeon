데이터 흐름을 설계하고 안정화하는 역할을 수행하고 있으며,  
데이터 엔지니어링 역량을 중심으로 커리어를 확장하고 있습니다.  

궁극적으로 개발은 문제를 해결하기 위한 수단이라고 생각합니다.  
이게 왜 문제인지, 어떻게 문제를 해결해야할지를 먼저 고민합니다.  
이런 마인드를 가지고 있어 특정 기술에 얽매이지 않습니다.  
항상 더 나은 방법을 선택할 수 있도록 지속가능한 학습을 늘 고민하고 실천합니다.
